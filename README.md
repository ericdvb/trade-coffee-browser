## Key Features

* Efficient Table View
  - View all products available from Trade in a single table based view
* Filter by any product attribute
* Search across all products

## How To Use

To run locally:
```bash
$ git clone git@github.com:ericdvb/trade-coffee-browser
```
```bash
$ yarn install
```
```bash
$ yarn start
```
